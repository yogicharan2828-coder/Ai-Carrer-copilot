# AI Career Copilot – Intelligent Resume Analysis & Career Guidance Platform

AI Career Copilot is a career guidance and resume analysis platform built using Flask, HTML, CSS, and JavaScript. The application helps users evaluate their resumes, receive structured feedback, identify improvement areas, and enhance overall job readiness through automated resume analysis and career recommendations.

## 🔍 How It Works

1. Users upload their resume.
2. The system processes and analyzes resume content.
3. Skills, education, projects, and profile strength are evaluated.
4. A resume score is generated based on predefined criteria.
5. Personalized feedback and improvement suggestions are provided.
6. Users receive actionable recommendations to improve job readiness.

## ⚙️ Installation

Clone the repository:

git clone https://github.com/yogicharan2828-coder/AI-Career-Copilot.git

Navigate to the project directory:

cd ai-career-copilot

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py

Open in your browser:

http://127.0.0.1:5000
